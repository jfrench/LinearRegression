9.1 Dictionary of Data

  • BMI18: the body mass index at age 18
  • WT2, WT9, and WT18: the weights at ages 2, 9, and 18 (in kg) for the \(n=70\) girls in the study.

9.2 Analysing Relations Between Regressors

# basic scatterplot matrix
pairs(~ BMI18 + WT2 + WT9 + WT18, data = BGSgirls)

  1. Based on the scatter plot matrix above, does there seem to be any relations among the regressors? Explain why these relations make practical sense.

  2. How can we adjust our model to account for the relations between the regressors?

9.3 Adjusting the Regressors

  1. Create a new regressor called DW9 that is the weight gain from age 2 to 9.

  2. Create a new regressor called DW18 that is the weight gain from age 9 to 18.

BGSgirls$DW9 <- BGSgirls$WT9-BGSgirls$WT2
BGSgirls$DW18 <- BGSgirls$WT18-BGSgirls$WT9
BGSgirls$DW218 <- BGSgirls$WT18-BGSgirls$WT2
# basic scatterplot matrix
pairs(~ BMI18 + DW9 + DW18, data = BGSgirls)

  1. Based on the scatter plot matrix above, how can you tell that the new regressors seem to be more independent from each other?
